# Smart Travel Planner (Static)

This repository contains a small two-section static UI (splash + home) under the `templates/` folder.

Quick start (no install required):

1. Serve with Python's simple HTTP server:

```bash
cd /workspaces/SMART-TRAVEL-PLANNER/templates
python3 -m http.server 8000
# open http://localhost:8000/combined.html
```

2. Or use the included npm dev script (requires Node.js/npm):

```bash
cd /workspaces/SMART-TRAVEL-PLANNER
npm run dev
# opens a local server serving `templates` on port 8000
```

Files of interest:
- `templates/combined.html` — single-file Splash + Home UI
- `templates/splash.html`, `templates/index.html` — alternate pages

Feel free to ask me to deploy this to GitHub Pages or create a temporary public tunnel.
